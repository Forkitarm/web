Fertigkeiten

03

Wordpress

WordPress ist ein kostenloses und Open-Source-Content-Management-System (CMS), das auf PHP und MySQL basiert. Es wird verwendet, um Websites, Blogs und andere webbasierte Anwendungen zu erstellen und zu verwalten. Es ist bekannt für seine Benutzerfreundlichkeit, Flexibilität und große Community-Unterstützung.

HTML, CSS und Javascript

JavaScript ist eine Programmiersprache, mit der interaktive und dynamische Webseiten erstellt werden können, die Dinge wie Formularvalidierung, Animationen und das Aktualisieren von Inhalten ermöglicht, ohne die Seite zu aktualisieren. Es wird normalerweise in Verbindung mit HTML und CSS verwendet.

HTML (Hypertext Markup Language) ist die Standard-Markup-Sprache, die verwendet wird, um Webseiten zu erstellen und die darauf angezeigten Inhalte zu strukturieren. Es definiert die Struktur und das Layout einer Webseite mithilfe einer Reihe von Tags und Attributen.

CSS (Cascading Style Sheets) ist eine Stylesheet-Sprache, die zur Beschreibung der Darstellung eines in einer Auszeichnungssprache geschriebenen Dokuments verwendet wird. Es wird verwendet, um Layout, Farben und andere visuelle Elemente von Webseiten zu steuern.

Grafikdesign, Adobe Creative-Cloud

Adobe Creative Cloud ist eine Sammlung von Software und Diensten für Grafikdesign, Videobearbeitung und Webentwicklung. Es umfasst beliebte Apps wie Photoshop, Illustrator und InDesign, mit denen Grafikdesigner professionelle visuelle Inhalte erstellen und bearbeiten können.

3D-Design, Blender

Blender ist eine kostenlose Open-Source-3D-Erstellungssoftware, die zum Erstellen von Animationen, Modellen, Kunst und Videospielen verwendet werden kann. Es bietet eine breite Palette von Werkzeugen für 3D-Modellierung, Texturierung, Beleuchtung, Animation und Compositing. Es ist auch eine beliebte Wahl für die Erstellung von Architekturvisualisierungen und Produktdesign.
